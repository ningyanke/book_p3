## 敏捷开发方法

### 敏捷开发

> * 以人为核心、迭代、循序渐进
> * 针对传统的瀑布开发模式的弊端
> * 分为多个相互联系、独立运行的小项目
> * 软件一直处于可使用状态

### 敏捷开发更符合软件开发的规律

> 软件---植物
>
> * 自底向上
> * 逐步有序
> * 遵循软件客观规律,
> * 迭代增量开发

### 轻量级软件开发方法

> * Scrum
> * 极限编程（XP）
> * 精益开发（Lean Development）
> * 动态系统开发方法（DSDM）
> * 特征驱动开发（Feature Driver develpment）
> * 水晶开发（Cristal Clear）
